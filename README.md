
# game adventure ex_6

This game about player that walk to rooms. In each rooms you can meet enemy or friend.

This module contain such classes:  
    -class Weapon  
    -class Bonus  
    -class Person  
    -class Friend(Person)  
    -class Enemy(Person)  
    -class Room  
    -class Field  
    -class Bag  
    -class Player  

Also this module have function that generates Field
