This module contain such classes:  
    -class Weapon  
    -class Bonus  
    -class Person  
    -class Friend(Person)  
    -class Enemy(Person)  
    -class Room  
    -class Field  
    -class Bag  
    -class Player  

Also this module have function that generates Field
